# AVL_Tree
##### 파이썬 프로그램을 실행하면 자동적으로 3가지 test가 진행됩니다.
## Test1
##### test1은 지정된 10개의 test할 data list와 지정된 5개의 delete할 data list를 통해서 AVL tree에 입력, 삭제, 검색이 잘 되는지 확인할 수 있습니다. 
#### Insert
##### 첫번째 줄에는 입력할 data와 삭제할 data를 보여준 뒤 입력을 시작합니다. 
##### 입력 실패는 입력이 완료된 후 검색을 통해 각 데이터들이 모두 잘 입력되었는지 확인한 뒤 입력되지 않은 값이 있다면 count합니다. 
##### 그래프 역시 inorder로 출력을 하여 가시적으로 프로그램이 알맞게 구현이 되었는지 확인할 수 있습니다. 
#### Delete
##### 삭제를 시작합니다. 
#### Check
##### 이후 삭제가 잘 되었는지 확인합니다. 
##### 만약 삭제한 데이터가 남아있는 경우를 count하여 출력해줍니다. 
##### 다시 한번 그래프를 inorder로 출력을 하여 가시적으로 프로그램이 알맞게 구현이 되었는지 확인할 수 있습니다. 
##### 마지막으로 남아있는 데이터를 모두 검색하여 알맞게 남아있는지 체크해줍니다.
## Test2
##### test2의 경우 random하게 선정된 100개의 test할 data list와 random하게 선정되는 50개의 delete할 data list를 통해서 AVL tree에 입력, 삭제, 검색이 잘 되는지 확인할 수 있습니다. 이후의 테스트 과정은 test1과 동일하게 진행합니다.
## Test3
##### test3의 경우 random하게 선정된 1,000,000개의 test할 data list와 random하게 선정되는 500,000개의 delete할 data list를 통해서 AVL tree에 입력, 삭제, 검색이 잘 되는지 확인할 수 있습니다. 
##### test3의 경우 data가 너무 많기 때문에 데이터 출력과 그래프 출력은 하지 않고 데이터 test1과 test2의 방법을 통해 입력되지 않은 데이터, 삭제되지 않은 데이터, 남아 있는 데이터를 체크해서 오류를 출력해줍니다.
